# Forest Cover Image Setup Instructions

## Steps to Add Your Forest Image as Cover Picture

### 1. Save the Forest Image
- Take the forest image you provided (the woman carrying items through the forest)
- Save it as `forest-cover.jpg` in the folder: `c:\Users\asus\Forest-Nexus\frontend\public\`
- Make sure the filename is exactly `forest-cover.jpg`

### 2. Verify the Changes
The code has been updated to use this image as the background for the hero section of your portal. 

### 3. What Was Changed
- **Home.js**: Updated the ParallaxHero component to use the forest image as background
- **App.css**: Added comprehensive styling for the hero section with proper overlay and responsive design

### 4. Features Added
- Beautiful parallax effect with your forest image
- Semi-transparent overlay to ensure text readability
- Responsive design that works on all devices
- Smooth animations and transitions

### 5. View the Changes
Once you save the image file, refresh your browser at `http://localhost:3000` to see the forest image as your portal's cover picture.

### 6. Image Specifications
- **Format**: JPG (recommended for photos)
- **Recommended size**: 1920x1080 or larger for best quality
- **File location**: `public/forest-cover.jpg`

The forest image will now serve as a beautiful, atmospheric background for your Forest Nexus portal, perfectly representing the forest environment and tribal context of your FRA monitoring platform.
